程序的debug
1. 光标移动到准备要暂停下来的可执行语句上;
2. 按Ctrl-F10,
则当程序执行到光标所在语句时会暂停下来;
3. 按F10则执行当前行语句;
另, 如果当前行是函数调用语句, 则按F11会进到该函数内执行(如果是库函数则不要进去了)。
(注意: 如果在停住之前执行了scanf()语句, 则需从键盘上输入, 程序会等待输入的!)

文件：
格式化文本读写：
FILE  *file;
file = fopen();
fscanf()
fprintf()
fclose()
句柄文件 常用于二进制读写：
int   f1;
f1 = _open()
_read()
_write()
_close()

"r" 打开，只读； "w" 打开，文件指针指到头，只写； 
"a" 打开，指向文件尾，在已存在文件中追加； 
"rb" 打开一个二进制文件，只读； 
"wb" 打开一个二进制文件，只写；
"ab" 打开一个二进制文件，进行追加 ；
"r+" 以读/写方式打开一个已存在的文件； 
"w+" 以读/写方式建立一个新的文本文件 ；
"a+" 以读/写方式打开一个文件文件进行追加 ；
"rb+" 以读/写方式打开一个二进制文件； 
"wb+" 以读/写方式建立一个新的二进制文件 ；
"ab+" 以读/写方式打开一个二进制文件进行追加 ；
当用fopen()成功的打开一个文件时，该函数将返回一个FILE指针，如果文件打开失败，将返回一个NULL指针。

文字和图形
// 定义本窗口(this)上作图对象
CClientDC   dc(this);
// 定义笔对象, 所有画线都是用笔的（笔风格：实线，笔尖宽：几个像素，笔颜色：（红，绿，蓝））
CPen   pen1( PS_SOLID, 1, RGB( 255, 0, 0 ) ),  *old_pen;
// 定义刷子对象,  对画区域的作图函数, 所有区域内的颜色都是用刷子颜色填充
CBrush   brush1( RGB( 0, 0, 255 ) ),   *old_brush;
// 对象选取函数
old_pen = dc.SelectObject( &pen1 );
old_brush = dc.SelectObject( &brush1 );//新对象选取，并返回老对象
dc.SelectObject( old_pen );
dc.SelectObject( old_brush );//老对象归位
